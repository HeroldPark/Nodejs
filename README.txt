1. Nodejs 디렉토리에서
    npm install express-generator

2. express ejs project 생성 - gocoder/views/index.ejs 생성 확인
    express -e gocoder

3. cd gocoder에서 모듈 설치(include ejs)
    npm install

4. 테스트
    node ./bin/www
    htpp://localhost:3000

5. nodemon 설치 및 시작
    npm install nodemon -g
    nodemon npm start

6. mysql 연동
    cd Nodejs/gocoder
    npm install mysql
